9P1A6874 A Bird a Box some Buddleia and a Bee.

Adrian Dancy posted a photo:

Black Redstart (Female) Phoenicurus ochruros, Manchester Site 2.

1833.27.10.2022


I photographed this Black Redstart in a private street in Manchester. There were buddlei plants insitu but I brought some Buddlei flower heads in from elsewhere to help create a bit of colour in some of my images. For a bit of fun I introduced an additional prop which is an old style model police box. It is in fact a bug box which I bought in a pet shop. Some may see the box as a mini Tardis which features in 'Dr Who' which is a famous kids TV series. I seem to recall they made a film too starring Peter Cushing as Dr Who. At the time this image was created the male and female Black Redstarts looked identical but I am certain that this was the female. The buddlei plant head attract many insects especially butterflies. On this occasion the flower head attracted a bee which caught the bird's attention.
